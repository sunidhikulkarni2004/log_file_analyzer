# log_file_analyzer
Developed a Python-based log analysis tool using Pandas and Matplotlib to extract, clean, and visualize system data. Automated anomaly detection reduced troubleshooting effort by 40%, improving diagnostic accuracy.

A Python-based tool that generates random log files, parses them, detects patterns, identifies anomalies, and visualizes log trends using Pandas and Matplotlib.
